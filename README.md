# 🌱 Sustainable Web Design


# 📖 Project Overview

**Sustainable Web Design** is a front-end web project that demonstrates how websites can be designed to be:

- ⚡ Fast
- 🌱 Environmentally Friendly
- ♿ Accessible
- 📱 Responsive
- 🎨 User-Friendly

The project focuses on reducing unnecessary digital waste by promoting lightweight design, optimized resources, accessibility best practices, and efficient user experiences.

---

# 🎯 Objectives

This project was created to:

✅ Promote sustainable web development practices

✅ Improve website performance and loading speed

✅ Reduce unnecessary data transfer

✅ Increase accessibility for all users

✅ Demonstrate modern UI/UX design principles

✅ Create awareness about environmentally responsible web design

---

# ✨ Key Features

## 🌙 Dark Mode Support

- Toggle between Light and Dark themes
- User preference stored using Local Storage
- Better viewing experience in low-light environments

---

## ♿ Accessibility Features

- High contrast design
- Keyboard focus indicators
- Proper semantic HTML structure
- Reduced motion support
- Readable typography and spacing

---

## 📱 Responsive Design

The website automatically adapts to:

- Mobile Devices
- Tablets
- Laptops
- Desktop Screens

using modern CSS Grid and Flexbox layouts.

---

## ⚡ Performance Optimization

The project demonstrates:

- Lightweight page structure
- Optimized images
- Reduced resource usage
- Efficient CSS styling
- Minimal JavaScript

---

## 📊 Sustainability Dashboard

The dashboard page provides sample sustainability metrics including:

| Metric | Description |
|----------|-------------|
| 📦 Total Page Size | Amount of data downloaded |
| 🌐 HTTP Requests | Number of resources loaded |
| ⚡ Load Time | Website loading speed |
| 🌱 CO₂ Emissions | Estimated carbon impact |
| 🏆 Eco Score | Sustainability rating |
