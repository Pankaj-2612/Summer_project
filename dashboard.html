Website Sustainability Dashboard

A sustainable website should not only look good, but also load quickly, use fewer resources, and reduce unnecessary data transfer. This dashboard shows the type of information a website owner can check to understand how eco-friendly and performance-friendly their website is.

What This Dashboard Shows

The values below are sample data for this project. In a real website, these details can be collected using performance testing tools and carbon estimation tools.

  • Total Page Size: Shows how much data the page downloads.
  • HTTP Requests: Shows how many files the browser loads.
  • Load Time: Shows how quickly the page opens for users.
  • CO2 Emission: Shows estimated carbon impact for a page view.
