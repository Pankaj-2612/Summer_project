UI / UX Features

These simple features improve usability, accessibility, and performance.

Dark Mode

Dark mode gives users a comfortable viewing option and can reduce screen brightness in low-light environments.

Good Contrast

Clear color contrast makes text easier to read for all users, including users with low vision.

Reduced Motion

Reduced motion support helps users who prefer less animation or are sensitive to moving effects.

Responsive Layout

The website adjusts to mobile, tablet, and desktop screens using flexible layouts.

Keyboard Focus

Visible focus styles help keyboard users see where they are on the page.
